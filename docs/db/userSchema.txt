Схема користувача:

Ім'я: Тип - Рядок, Обов'язково. Опис: Ім'я користувача.

Прізвище: Тип - Рядок, Обов'язково. Опис: Прізвище користувача.

Електронна пошта: Тип - Рядок, Обов'язково, Унікально. Опис: Електронна пошта користувача.

Пароль: Тип - Рядок, Обов'язково. Опис: Пароль користувача.

Адреса: Тип - Рядок. Опис: Адреса користувача.

Телефон: Тип - Рядок. Опис: Номер телефону користувача.

Дата реєстрації: Тип - Дата/час. Опис: Дата та час реєстрації користувача.

Адміністратор: Тип - Булівське. Опис: Статус адміністратора (чи є користувач адміністратором).

Активний: Тип - Булівське. Опис: Статус активності облікового запису користувача.
